All components are imported flat from `@hareru/ui`:

```tsx
import { Button, Card, Dialog, toast } from "@hareru/ui"
```

> This is a guide to the main components. For the full public API including subcomponents, variant functions, and type exports, see [src/index.ts on GitHub](https://github.com/hareru-ai/ui/blob/main/packages/ui/src/index.ts).
