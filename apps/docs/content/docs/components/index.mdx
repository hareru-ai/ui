All components are imported flat from `@hareru/ui`:

```tsx
import { Button, Card, Dialog, toast } from "@hareru/ui"
```

## Component Patterns

### A. Standalone Components

Basic elements that operate with props alone.

| Component | Primary Use |
|-----------|------------|
| [Button](/docs/components/button) | Button with variant and size props |
| [Badge](/docs/components/badge) | Status label |
| [Input](/docs/components/input) | Text input |
| [Textarea](/docs/components/textarea) | Multi-line text input |
| [Label](/docs/components/label) | Form label |
| [Separator](/docs/components/separator) | Divider line |
| [Skeleton](/docs/components/skeleton) | Loading placeholder |
| [Switch](/docs/components/switch) | Toggle switch |
| [Progress](/docs/components/progress) | Progress bar |
| [Toggle](/docs/components/toggle) | Toggle button |

### B. Composite Components

Used by combining multiple sub-components.

| Component | Sub-components |
|-----------|---------------|
| [Card](/docs/components/card) | CardHeader, CardTitle, CardDescription, CardContent, CardFooter |
| [Alert](/docs/components/alert) | AlertTitle, AlertDescription |
| [EmptyState](/docs/components/empty-state) | EmptyStateIcon, EmptyStateTitle, EmptyStateDescription, EmptyStateAction |
| [FormField](/docs/components/form-field) | FormFieldLabel, FormFieldControl, FormFieldDescription, FormFieldMessage |
| [Tabs](/docs/components/tabs) | TabsList, TabsTrigger, TabsContent |
| [BentoGrid](/docs/components/bento-grid) | BentoGridItem |
